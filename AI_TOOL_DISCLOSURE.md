# AI Tooling Disclosure

GemmaJudge was built and verified by the team during AMD Developer Hackathon: ACT II.
The following third-party AI tools assisted the work:

- OpenAI-based coding assistants were used for implementation support, code review,
  debugging, tests, documentation, and submission copy. Relevant commits also carry an
  `OpenAI <noreply@openai.com>` co-author trailer where applicable.
- Gamma was used to compose and visually design the slide presentation from team-provided
  project content.

The team selected the product direction, controlled all accounts and credentials, reviewed
the generated material, ran the tests, performed the AMD GPU runs, and checked every public
metric against committed artifacts. AI-generated suggestions are not treated as evidence;
the repository logs, JSON reports, screenshots, and reproducible commands are the evidence.

No secret, private customer data, or third-party proprietary source material was supplied
to these tools for this submission.
